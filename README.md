# news-nova
NewsNova is a Laravel-based news application that integrates with external APIs to fetch and display real-time news. It uses JWT authentication and follows service-based architecture with dependency injection.

## Features
- JWT Authentication
- Fetch real-time news from external API
- Search functionality
- Clean service-based architecture

## Tech Stack
- Laravel
- React
- MySQL
- JWT Auth

## Project Structure
- /backend → Laravel API
- /frontend → React UI

## Setup Instructions
1. Clone repo
2. Setup backend (.env, composer install, php artisan serve)
3. Setup frontend (npm install, npm start)
